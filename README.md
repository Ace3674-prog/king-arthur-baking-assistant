# King Arthur Baking Assistant 🍞

An AI-powered baking assistant that helps users find products, get recipes, and answer baking questions.

## Features
- Product search and recommendations
- Baking tips and advice
- Recipe suggestions
- Price filtering

## Note
This app uses Ollama for local LLM inference. For Hugging Face Spaces deployment, the model runs on CPU.
