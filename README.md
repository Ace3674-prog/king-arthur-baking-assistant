# King Arthur Baking Assistant 🍞

An AI-powered baking assistant powered by LangGraph and Ollama.

## Features
- Product search and recommendations
- Recipe suggestions
- Baking tips and advice
- Price filtering

## Note
This app uses Ollama for LLM inference. Make sure Ollama is running locally.
